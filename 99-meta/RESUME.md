## The pipeline (all reusable; scripts in `C:\Users\Josep\AppData\Local\hermes\scripts\`)

1. **Build the queue:** canonicalize every URL to `https://youtu.be/<id>` (prefetch regex does NOT match `/shorts/`; metadata regex does). Dedup with the loop: `for id in $ids; do grep -rqF -- "$id" Sources/ && echo BUILT || echo NEW; done`.
2. **Prefetch transcripts:** `prefetch-transcripts.py <N> <queuefile>` → `scripts\transcripts\<vid>.txt`. **A fresh Proton IP yields ~16–22 fetches before YouTube blocks it**; block is often short-lived (~70s) — re-run on the SAME IP recovers the tail. When an IP stays blocked, **rotate to a fresh/small-country server** (Belarus/small countries & P2P least blocked; US/Canada re-block fast). `yt-dlp` is NOT a fallback (429/bot-check). Put low-priority vids EARLY (last-in-queue eat the limit). oEmbed `get-yt-meta.py --cached-only` works under a transcript block. Caption-disabled videos return "Subtitles are disabled" — unbuildable, skip.
3. **Build (parallel):** spawn general-purpose agents, one per cached vid, in waves of ~7 (drain between waves; stagger same-topic videos so two concurrent agents don't create the same new note). ⚠ **VPN CAUTION:** the whole machine routes through Proton, so a VPN blip kills in-flight build agents (socket/ECONNRESET errors → re-spawn). For big rounds, **prefetch ALL transcripts first, then build with the VPN left stable or disconnected** (builds need no internet). Each agent: READ the cached transcript (no internet), write deep notes to FLAT folders. Rules: tags lowercase-hyphenated-NO-SPACES; **SINGLE-bracket** cites `[^n]: [[Source|Title]], [MM:SS], <url>` (NEVER `[[MM:SS]]` — orphan nodes; sweep `grep -rlE '\[\[[0-9]{1,3}:[0-9]{2}'` must be 0); ≥2 wikilinks/note; existing Scripture/People → wikilink only, don't edit; foil/skeptic sources get `## Apologetic Engagement`/`## LDS Relevance`; **NEVER touch** `99-meta\`,`MOCs\`,`Glossary\`,`Sources Index.md`, other sources. Glob fails from agent cwd → use Bash `ls`/Grep with ABSOLUTE paths + FORWARD SLASHES (trailing `\` before `"` breaks the quote). After a big run, **gap-check**: loop all queue IDs, grep `Sources/`, re-spawn any missing.
4. **Books/articles (no internet):** `get-pdf-text.py <pdf> <first> <last>` → `[p.N]` markers; `get-article.py <url>` → clean text (trafilatura). Small docs: one agent (or yourself) → Source + concepts, cite `p.NN`/url. Big books (e.g. Ostler 70pp): one thorough agent or chapter agents.
