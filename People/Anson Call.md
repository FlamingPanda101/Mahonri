# Anson Call

Early Latter-day Saint pioneer whose biography is a key source for the [[Rocky Mountains Migration Prophecy]].

## Source of the Rocky Mountains Prophecy
Per a FAIR article the host cites, the account of [[Joseph Smith]] prophesying that the Saints would go to the Rocky Mountains traces to Anson Call's biography, set in August 1842 (the Montrose/Nauvoo period). The host treats Call's account — alongside Benjamin F. Johnson and others — as part of the multiple attestation that makes it "very likely" Joseph did speak of the Saints going west. See [[Multiple Attestation - Corroborating Apocryphal Sources]] and [[Mormonism with the Murf - Joseph Smith's Top 20 Fulfilled Prophecies]].[^1]
