# Dr. Julia Shaw

Criminal psychologist and memory researcher who studies how false memories form and how they corrupt criminal testimony.

## The AI-hallucination analogy for memory
Shaw compares human memory to artificial intelligence: when given a prompt, AI "assembles what it supposes the answer should be, even if it has to assemble it from irrelevant or completely false components. The brain acts in a similar manner" — a false memory forms when a suggestive event is incorporated into existing memories.
