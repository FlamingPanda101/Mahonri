# John 17:3

> And this is life eternal, that they might know thee the only true God, and Jesus Christ, whom thou hast sent. *(KJV)*

## Key Term: "the only true God" (*ton monon alethinon theon*)
In Jesus' high-priestly prayer, "the only true God" is grammatically the *Father* ("thee"), set alongside "and Jesus Christ whom thou hast sent." The verse is a standing prooftext in debates over Trinitarian consubstantiality: a plain reading distinguishes the Son from "the only true God," while classical exegesis takes "only true God" to exclude *idols*, with the Son included as co-object of saving knowledge (cf. [[John 1.1|John 1:1]], [[John 20.28|John 20:28]]).
