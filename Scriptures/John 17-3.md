# John 17:3

> "And this is life eternal, that they might know thee the only true God, and
> Jesus Christ, whom thou hast sent."

**Apologetic issue**: Used to claim Jesus identifies the Father alone as "the only
true God," excluding LDS teachings about exalted beings or a divine council.

**Rebuttal**: [[John 17-3 - Only True God]]
