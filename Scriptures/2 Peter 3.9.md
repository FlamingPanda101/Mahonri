# 2 Peter 3:9

> The Lord is not slack concerning his promise, as some men count slackness; but is longsuffering to us-ward, not willing that any should perish, but that all should come to repentance. *(KJV)*

## Key phrase
"**Not willing that any should perish**" — Greek **μὴ βουλόμενός τινας ἀπολέσθαι**. Paired with [[1 Timothy 2.4|1 Tim 2:4]] as the second pillar of the universal-salvific-will argument.

## Apologetic application
[[Gavin (Catholic Debater)]] cites it to show God's benevolence is universal "even when salvation is not universally accepted." See [[Scripture Explicitly Affirms Universal Salvific Will]]. Hayden's rebuttal: a *desire* that none perish does not resolve the prior act of freely creating those one foreknows will perish. See [[The Creation Dilemma]].
