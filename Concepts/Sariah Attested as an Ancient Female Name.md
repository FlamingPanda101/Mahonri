# Sariah Attested as an Ancient Female Name

## The Claim
**Sariah** — the name of Lehi's wife and one of only a few named women in the Book of Mormon — is now **attested as a genuine ancient Hebrew female name** in the **Elephantine** documents (the 5th-century-BC Jewish community in Egypt), including a complete attestation found by [[Neal Rappleye]]. [^1]

## Supporting Evidence
1. **Two Elephantine attestations.** *Sariah* appears twice in the Elephantine corpus. One was **incomplete/broken**, which left [[Paul Hoskisson]] wanting a fuller attestation rather than a reconstruction. Strength: **moderate** (for the partial). [^1]
2. **A complete attestation (the key find).** [[Neal Rappleye]] located an **ostracon** (a list of supplies/rations) reading "**Sariah daughter** [of ...]"; though it broke before the father's name, it preserves *both* the full name **Sariah** and the word **daughter**, confirming the name *and* its use for a female. Strength: **high** (a complete, gender-confirming attestation). [^2]
3. **Methodological significance.** It converts Sariah from a reconstruction into an attested name, the stronger level-1 evidence (see [[Attestation Versus Wordplay - Two Levels of Evidence]]). Strength: **high**. [^2]

**Steelman:** A skeptic notes that a single ostracon from Elephantine (a Jewish colony *in Egypt*, centuries after Lehi) only shows the name *could* exist in some Jewish circles, not that it was current in pre-exilic Jerusalem; and that Hebrew female names built on *sar-* ("princess," cf. Sarah) are common enough that an attestation is unsurprising. The reply: even granting commonness, having *Sariah* specifically attested removes the prior critic complaint that it was a made-up or reconstructed-only name.

## Best Quotes
> "In the Elephantine documents, it's actually attested twice ... in an incomplete form ... and that caused some — like Paul Hoskisson would say, it's nice that we have it reconstructed, but we'd like to have the actual attestation." — [[Matthew Bowen]] [^1]

> "We do have an attestation on an ostracon ... it had Sariah daughter ... it has both the full Sariah and the full daughter. So we know it is that name and we know it's a female." — [[Neal Rappleye]] [^2]
