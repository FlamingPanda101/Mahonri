# Faithkeepers Would Return if a Friend Invited Them

## The Claim
Because most who leave drift away for logistical or relational reasons rather than rejecting belief, a large share say they would return if simply invited by a friend — making them low-effort, high-yield reactivation candidates.

## Supporting Evidence

1. **Self-reported willingness to return (rated: moderate, survey).** "A lot of them said that they would return if a friend invited them," per the data Dyer cites and the accompanying Deseret News article.[^1] The host frames this as "low-hanging reactivation fruit" reachable through ministering brothers and sisters.[^2]

2. **Hinckley's three-part formula (rated: moderate, doctrinal application).** The principle maps onto President Gordon B. Hinckley's counsel that every new member needs "a friend, a responsibility [calling], and nurturing with the good word of God" — the same retention ingredients apply to reactivation.[^3] See [[Gordon B. Hinckley]].

3. **Belief is intact, so the barrier is relational (rated: moderate).** Since faithkeepers and faith-leaners retain belief and warm feelings (see [[Most Who Leave Drifted Away Rather Than Rejected Faith]]), the missing piece is connection, not conversion.[^1] *Steelman:* "I would return if invited" is a cheap survey affirmation that may overstate real-world follow-through; intention-behavior gaps are large, and an invitation alone rarely overcomes the life circumstances that caused the drift.

## Best Quotes

> "A lot of them said that they would return if a friend invited them."[^1]

> "We've got some low-hanging reactivation fruit there... we just need to get those ministering brothers and sisters out there."[^2]

> "President Hinckley said... every single new member needs a calling, nurturing the good word of God, and a friend. And so it sounds like for those who leave the church, those same principles really do apply."[^3]
