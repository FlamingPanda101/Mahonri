# Unlimited Chances Destroy Probability - The Royal Flush Objection

## The Point
Hansen's sharpest objection to the multiverse is not empirical but methodological: positing an unlimited number of universes "destroys any notion we have about probability because it makes anything that is possible actual."[^1] If every possibility is realized somewhere, then no observation can ever count as surprising or as evidence for anything — which Hansen argues is a *reductio*: it would equally explain away a cheater. He dramatizes this with a poker player who draws royal flushes on every hand for two hours and protests, "I'm not cheating... there are just nearly infinite universes out there and we just happen to be in the one where I get all royal flushes."[^2] Seth's rejoinder is that cosmic *selection* (only black-hole-compatible universes survive) is not unconstrained chance, so "not all possibilities arise — only the possibilities compatible with selecting for black holes."[^3]

## Evidence
1. **The mechanism proves too much.** "Want to get a hundred royal flushes in a row? Just play an infinite number of hands... eventually it will happen. Don't have flying unicorns in this universe? Well if there are infinite universes you will have them in one of those universes."[^1] Hansen extends it: on the atheist's logic there is even "a universe in which the church is true," because "everything that is possible will be actual."[^4]
2. **The multiverse is a post-hoc dodge.** "The multiverse is a clear post-hoc invention that people made up to avoid the obvious evidence for design... if evidence for design did not exist, the atheist would not have postulated multiple universes."[^5] In cross-examination Seth concedes there is no *direct* evidence of any other universe.[^6] See [[Multiverse as Ad-Hoc Escape from Fine-Tuning]].
3. **Occam's razor favors a mind.** Hansen frames the choice as Occam's razor: faced with "a set of 100 royal flushes in a row at the base of physics," the simpler explanation given "our uniform experience" is an intelligent mind, not an infinite ensemble of chances.[^7] He answers the "but a mind is complex" objection with the safecracker: it is *less* complex to infer that someone gave the burglar a 15-digit code than that he hit it "by random chance."[^8] See [[A Designer Would Be as Complex as the Universe]].
4. **Seth: constrained selection, not raw chance.** Seth replies that the black-hole model is a *selection pressure*, so "not all possibilities arise"; the objection that the multiverse "erases all probability" misfires because the ensemble is filtered, not random.[^3]

## Best Quotes
> "Appealing to unlimited chances destroys any notion we have about probability because it makes anything that is possible actual." (Hansen) [^1]

> "Imagine that person saying I'm not cheating... there are just nearly infinite universes out there and we just happen to be in the one where I get all royal flushes on every hand." (Hansen) [^2]

> "If we are to grant this level of selection pressure where only the universe compatible with black holes can exist... not all possibilities arise, only the possibilities compatible with selecting for black holes." (Seth) [^3]

## LDS Connection
This is a general probabilistic-reasoning argument with no distinctively LDS content; its apologetic value is method-level — it presses the atheist to show that the universe-generating ensemble is *constrained* rather than an unconstrained "anything-goes" device, which is precisely the burden taken up in [[The Multiverse Cannot Escape Prior Fine-Tuning]].
