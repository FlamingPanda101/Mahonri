# Covenant Code Lexical Precision on Social Status

## The Claim
[[Dan McClellan]] argues the Covenant Code (Exodus 21) is "very peculiar but also consistent in the language it uses" — it deploys distinct Hebrew terms for each social category because **punishments differ by the social status of perpetrator and victim**. This lexical precision is decisive for reading the slave laws correctly.[^1]

## Supporting Evidence

1. **The term inventory (rated: strong; linguistic).** *ish* = native Israelite man; *isha* = woman; *ben* / child-of-an-*ish* = child; a foreigner is named as a foreigner; an enslaved person is an *eved*. "It is very clear about the references to perpetrators and victims."[^1]

2. **Status determines penalty (rated: strong; linguistic-legal).** Because the punishment varied by who did what to whom, the precise term used in each law tells us whom it protects. [[Exodus 21.16|Exodus 21:16]]'s use of *ish* shows it protects native Israelite men.[^1]

3. **Resolves the "avenged" ambiguity (rated: moderate).** In [[Exodus 21.20-21|Exodus 21:20-21]], McClellan says the punishment for beating a slave to death is unspecified precisely because the law does not distinguish whether the victim was an Israelite debt slave or a foreign chattel slave — the penalty would differ accordingly.[^2]

4. **Steelman.** One could argue *ish* sometimes means "person" generically, broadening the law's protection. McClellan's reply is that within the Covenant Code's deliberately differentiated vocabulary, the generic reading is unwarranted — the author had distinct words available and used them.[^1]

## Best Quotes

> "When it talks about an *ish* it's talking about a native Israelite man. If it's talking about a woman it's using *isha*. If it's talking about a child it's using *ben*... If it's talking about an enslaved person it refers to an *eved*. It is very clear about the references to perpetrators and victims because the punishments would differ depending on the social status of the perpetrator and/or the victim."[^1]
