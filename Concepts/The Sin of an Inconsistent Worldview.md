# The Sin of an Inconsistent Worldview

## The Point
[[Jacob Hansen]] distinguishes the *logical* sense of (in)consistency from the everyday one. In ordinary speech, "consistent" means reliable or predictable. In logic, consistency is a property of a *set of claims*: the set is consistent only if it is logically possible for all of them to be true at the same time — that is, if it does not entail a contradiction.[^1] A contradiction has the form "a and not-a" and is "false in all logically possible worlds."[^2] So a worldview is *incoherent* when its own claims contradict one another, and — crucially — calling a view incoherent "does not mean that you simply don't understand it or like it"; it is to say "the position is contradictory and cannot be true," like asserting that 2 + 3 = 7.[^3] This is the "intellectual sin" the video targets: holding a set of beliefs that cannot all be true together. Compare [[Not Impossible Is Not the Standard for Contradiction]].

## Evidence
1. **The logical definition.** "In logic, consistency is a property of sets of claims. We say that a set of claims is consistent if it's logically possible for all of them to be true at the same time," where logically possible means "the set of claims doesn't entail a logical contradiction."[^1]
2. **Worked contradiction.** "All humans are mortal" and "some humans are not mortal" form an inconsistent set: they are logical contradictories, so "if one is true the other must be false." Adding hidden premises (e.g., "Simon is human" to "all humans are mortal" + "Simon is immortal") can turn a consistent set inconsistent.[^4]
3. **Definition of an incoherent worldview.** "A person's worldview is inconsistent or incoherent if the claims of that worldview contradict each other."[^2]
4. **The disqualifying force.** Internal contradiction is not a minor defect: a contradictory worldview is "disqualified from the start... by the very laws of logic itself." See [[Logical Consistency Is a Necessary Condition for Truth]].[^5]

## Best Quotes
> "Saying something is logically incoherent or inconsistent does not mean that you simply don't understand it or like it. It is instead to say that the position is contradictory and cannot be true." [^3]

> "A person's worldview is inconsistent or incoherent if the claims of that worldview contradict each other." [^2]

## LDS Connection
The charge of incoherence is a double-edged tool that the believer must also submit to: Hansen's payoff is precisely that *his own* worldview, "while it has its challenges, at least it's relatively consistent."[^5] For a Latter-day Saint, this reframes apologetics as a test of internal coherence applied evenhandedly — the same standard the critic uses must be borne by the critic. Restoration epistemology already affirms multiple convergent witnesses to truth ([[Scientism Is Self-Refuting]]), but it does not exempt faith claims from non-contradiction; rather, it invites believers to examine rival worldviews "critically" instead of perpetually playing defense. The structural move — exposing a position that cannot satisfy the very logic it depends on — is the same one used against [[Bill Reel's Free-Will Double Standard - Determinism Cannot Ground Moral Blame|Bill Reel's free-will double standard]].
