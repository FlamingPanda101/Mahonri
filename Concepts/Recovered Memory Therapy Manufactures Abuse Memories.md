# Recovered Memory Therapy Manufactures Abuse Memories

## The Claim (Objection Rebutted)
"Recovered memory therapy" can unlock genuine, previously inaccessible childhood memories of abuse — so the satanic-ritual-abuse memories it surfaced in the 1980s should be believed.

## The Rebuttal

### 1. It was an experimental, unproven technique (strong)
[[Lawrence Pazder]] used "an experimental new therapy" on [[Michelle Smith]] to "unlock previously unknown and inaccessible memories." The technique's debut product, *Michelle Remembers*, produced claims for which "no evidence has ever been found to corroborate" [^1][^2]. See [[Michelle Remembers Sparked the Satanic Panic]].

### 2. Memory is reconstructive, not a recording (strong; cites research)
"Memory is a reconstructive process that can be prone to errors or distortions during retrieval." False memories "are not necessarily lies, and they aren't always intentionally produced" — the person recites them "with complete sincerity and innocence" [^3][^4]. Researchers at Boston College tied susceptibility to false memory to brain deficiencies [^3].

### 3. The AI-hallucination analogy (strong; expert framing)
Criminal psychologist [[Julia Shaw]] compares memory to artificial intelligence: "when given a prompt, AI assembles what it supposes the answer should be, even if it has to assemble it from irrelevant or completely false components. The brain acts in a similar manner" — a false memory forms when a suggestive event is incorporated into existing memories [^4].

### 4. False memories have been deliberately implanted (decisive)
[[Elizabeth Loftus]] "recreated false memories and successfully implanted them in people's brains," coaching subjects into recalling events that "demonstrably never happened" (the "lost in the mall" experiment) [^5]. See [[False Memory Syndrome Explains the SRA Accusations]].

### 5. The technique became an industry then a tort (moderate)
*Michelle Remembers* and Pazder's seminars trained police and social workers [^6]; today "false memory syndrome" (FMS) is grounds for malpractice lawsuits against therapists, cataloged by the FMS Foundation, with the list "growing every day" [^7].

## Best Quotes
> "Memory is a reconstructive process that can be prone to errors or distortions during retrieval. False memories are not necessarily lies, and they aren't always intentionally produced." [^3]

> "Dr. Elizabeth Loftus has actually recreated false memories and successfully implanted them in people's brains." [^5]
